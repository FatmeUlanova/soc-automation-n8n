# SOC Automation with n8n

This repository contains cybersecurity automation workflows built with n8n.

## Workflow 01 - SOC Alert Simulator

Detects possible brute-force activity using a simulated Windows Event ID 4625.

## Workflow 02 - Threat Intelligence Lookup

Takes an IP address, performs GeoIP enrichment, and classifies risk using IF logic.

## Tools Used

- n8n Cloud
- HTTP Request node
- IF node
- Edit Fields node
- ip-api.com
- GitHub

## Skills Demonstrated

- Security automation
- API usage
- JSON processing
- Conditional logic
- SOC alert enrichment
